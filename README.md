# rest-todo-api

REST Todo API: A simple node/express API exercise.

### Quick start

```bash
# clone repo
$ git clone https://github.com/juansoos/rest-todo-api

# change directory to cloned app
$ cd rest-todo-api

# install the dependencies with npm
$ npm install

# start dev server (with live-reload)
$ npm start
```
